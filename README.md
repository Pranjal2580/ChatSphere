# ChatSphere – Full-Stack Real-Time Chat Application

## Overview

ChatSphere is a real-time chat application built using the MERN stack and Socket.io. It enables seamless one-to-one and group communication with instant message delivery, media sharing, and secure authentication.

## Features

* Real-time one-to-one messaging
* Group chat functionality
* Online/Offline user status
* Typing indicators
* Image, audio, and video sharing
* Voice message support
* JWT-based authentication and authorization
* Secure password hashing with bcrypt
* Responsive and user-friendly interface

## Tech Stack

### Frontend

* React.js
* Context API
* Axios
* Socket.io Client

### Backend

* Node.js
* Express.js
* Socket.io
* JWT Authentication

### Database

* MongoDB
* Mongoose

## Installation

### Clone Repository

```bash
git clone https://github.com/Pranjal2580/ChatSphere.git
```

### Backend Setup

```bash
cd backend
npm install
npm start
```

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```
